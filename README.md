## 📊 Indonesia Payment System Dashboard

This project presents an analytical dashboard examining the evolution of Indonesia’s payment system, with a focus on QRIS adoption, digital payment growth, and financial inclusion disparities across regions.

Using a combination of BIS CPMI-style payment statistics and World Bank financial inclusion frameworks, the analysis highlights structural shifts from cash-based to digital transactions, as well as regional gaps in access and usage.

Key features include:
- Time-series analysis of QRIS growth (merchants, users, and transaction value)
- Comparative payment trends across instruments (mobile, card, e-money, ATM)
- A GeoPandas-based choropleth map visualizing financial inclusion at the provincial level
- Regional benchmarking of digital payment penetration

This project is designed to simulate a payment system surveillance use case, supporting data-driven insights for policymakers and financial institutions in monitoring system efficiency, inclusion, and digital transformation.
